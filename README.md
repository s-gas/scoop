# scoop

WIP

## Description

Command-line tool that displays the directory structure using indentation to show hierarchy.

## How to run

```bash
./scoop [flags] [path]
```

### Available flags:

- Maximum depth (default 4)
```bash
--depth int
```

- Show hidden files
```bash      
--hidden
```

- Highlight files containing given string
```bash
--find string
```
